# Angles Proto v0.10

React SPA for managing hold angles (MAIN + STEFAN tables).

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Admin Panel

Navigate to `/#/admin`
- First login: `admin` / `admin` (hash saved to localStorage)

## Build & Deploy

```bash
npm run build
vercel --prod
```
